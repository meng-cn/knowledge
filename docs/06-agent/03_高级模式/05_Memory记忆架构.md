## 1. Agent 的三层记忆

```
┌─────────────────────────────────┐
│       Agent Memory              │
│                                 │
│  ┌───────────┐ ┌───────────┐   │
│  │ 工作记忆   │ │ 短期记忆   │   │

│  │ 当前任务   │ │ 对话历史   │   │
│  └───────────┘ └───────────┘   │
│                                 │
│  ┌───────────┐ ┌───────────┐   │
│  │ 长期记忆   │ │ 个人记忆   │   │
│  │ (Long-term)│ │ (Personal)│   │
│  │ 知识库     │ │ 用户偏好   │   │
│  └───────────┘ └───────────┘   │
└─────────────────────────────────┘
```
