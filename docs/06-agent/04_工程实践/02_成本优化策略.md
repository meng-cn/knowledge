## 1. 五大优化策略

### 1.1 模型路由（按任务分级）

| 层级 | 模型 | 成本/1M tokens | 适用 |
|--|-|--|- |
| L1（简单，60-70%） | GPT-4o-mini / Claude Haiku | $0.15-0.50 | 简单任务 |
| L2（中等，20-30%） | Claude Sonnet / GPT-4o | $3-15 | 中等任务 |
| L3（复杂，5-10%） | GPT-4.5 / Claude Opus | $15-60 | 复杂推理 |

### 1.2 缓存策略

缓存是 Agent 成本优化中 ROI 最高的手段，可消除 20-45% 的 API 调用。[来源: AgentWiki - Caching Strategies for Agents](https://agentwiki.org/caching_strategies_for_agents)

### 1.3 提示词优化

Few-shot learning 可保持同等效果的同时减少 20-40% token 消耗。[来源: AI Cost Optimization (AIToolsBusiness)](https://aitoolsbusiness.com/ai-cost-optimization/)

### 1.4 Prompt 缓存

企业环境中，prompt caching（复用系统 prompt 的 KV cache）可大幅降低重复请求成本。[来源: Inductivee - LLM Cost Optimization](https://inductivee.com/blog/enterprise-llm-cost-optimization)
