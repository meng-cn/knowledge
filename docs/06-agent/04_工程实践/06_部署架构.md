## 1. 部署架构

### 1.1 三种部署模式

| 模式 | 场景 | 示例 |
|------|------|------|
| **Serverless** | 轻量 Agent | Cloudflare Workers, Vercel |
| **容器化** | 标准部署 | Docker + K8s |
| **边缘部署** | 低延迟 | Edge Workers, Wasm |

### 1.2 推荐架构

```
客户端 → API Gateway → Agent Service → LLM Provider
                    → Vector DB
                    → MCP Servers
```

### 1.3 快速部署示例

```python
# Vercel Edge Function
import { Configuration, OpenAIApi } from "openai";
import { AgentsSDK } from "openai-agents";

export default async function handler(req, res) {
  const agent = new AgentsSDK({ apiKey: process.env.OPENAI_API_KEY });
  const result = await agent.run(req.body.message);
  res.status(200).json(result);
}
```
