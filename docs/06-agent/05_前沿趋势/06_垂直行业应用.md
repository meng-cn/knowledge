# 垂直行业应用

> **在知识图谱的位置**：模块五 · 05_前沿趋势 · 第 6 节
> **难度**：⭐⭐ | **前置知识**：Agent 基础

---

## 1. 概述

Agent 从通用走向垂直行业，是 2025 年最重要的落地趋势。

---

## 2. 主要行业应用

### 2.1 金融科技

| 场景 | Agent 能力 |
|------|------|
| 智能客服 | 自动回答、投诉处理 |
| 风控 Agent | 实时风险检测 |
| 投研 Agent | 自动研报生成 |
| 合规 Agent | 自动审查 |

### 2.2 医疗健康

| 场景 | Agent 能力 |
|------|------|
| 诊断辅助 | 影像分析 + 文献检索 |
| 药物发现 | 分子生成 + 筛选 |
| 健康管理 | 个性化健康建议 |

### 2.3 教育

| 场景 | Agent 能力 |
|------|------|
| 个性化教学 | 自适应学习路径 |
| 作业批改 | 自动评估 |
|  Tutor Agent | 一对一辅导 |

### 2.4 代码开发

| 场景 | Agent 能力 |
|------|------|
| 代码生成 | Copilot, Cursor |
| 代码审查 | 自动 PR Review |
| Bug 修复 | 自动修复 |
| 文档生成 | 自动生成 |

### 2.5 电商

| 场景 | Agent 能力 |
|------|------|
| 智能推荐 | 个性化推荐 |
| 智能客服 | 自动应答+售后 |
| 商品上架 | 自动上架+优化 |
| 供应链优化 | 预测+调度 |

---

## 3. 垂直 Agent 设计模式

```
垂直 Agent = 通用 Agent 能力 + 行业知识 + 行业工具
```

| 要素 | 说明 |
|------|------|
| **行业知识** | 领域术语、流程、规则 |
| **行业工具** | 行业 API、数据库 |
| **行业模板** | Prompt 模板、工作流 |
| **合规要求** | 行业合规、数据安全 |
