## 1. 概述

**Pydantic AI** 是 Pydantic 团队推出的 Python 原生 Agent 框架，利用 Pydantic 的类型系统和验证能力，让 Agent 的工具调用**类型安全、零运行时错误**。

核心理念：**类型安全 = 更少的 Bug + 更好的 DX**
