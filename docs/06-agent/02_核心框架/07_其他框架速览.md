# 其他框架速览

> **在知识图谱中的位置**：模块二 · 02_核心框架 · 第 7 节
> **难度**：⭐⭐ | **前置知识**：其他框架

---

## 1. CrewAI — 多 Agent 团队

CrewAI 将多个 Agent 组织成"团队"，每个 Agent 有角色、目标和工具。原生支持 MCP Server。

- **文档**：[CrewAI](https://docs.crewai.com/)
- **MCP 集成**：[CrewAI MCP DSL](https://docs.crewai.org.cn/en/mcp/dsl-integration)

## 2. AutoGen — 多 Agent 对话（已与微软 Agent Framework 合并）

Microsoft AutoGen 通过 Agent 对话完成任务，原生支持 MCP。

- **文档**：[AutoGen](https://microsoft.github.io/autogen/)
- **AutoGen + MCP**：[AutoGen与MCP文章](https://llmmultiagent.com/blogs/autogen_mcp_blog)

## 3. MetaGPT — 多 Agent 软件工程

用多 Agent 模拟软件公司的完整工作流。

- **论文**：[MetaGPT 论文](https://arxiv.org/abs/2308.00352)

## 4. Phidata — 本地优先 Agent

开源 Agent 框架，强调本地运行和自定义。

- **GitHub**：[Phidata](https://github.com/phidatahq/phidata)

## 框架选型对比

| 框架 | 范式 | 核心优势 | 适合 | 来源 |
|--|-|--|-|--|
| OpenAI Agents SDK | 轻量 Agent | 官方 SDK | 快速开发 | [GitHub](https://github.com/openai/openai-agents-python) |
| LangGraph | 有向图 | 复杂工作流 | 生产级 | [LangChain](https://langchain-ai.github.io/langgraph/) |
| LangChain | 链式 | 生态大 | 简单链式 | [LangChain](https://python.langchain.com/) |
| LlamaIndex | 检索优先 | RAG | 知识库 | [LlamaIndex](https://docs.llamaindex.ai/) |
| CrewAI | 多Agent团队 | 角色分工 | 团队协作 | [CrewAI](https://docs.crewai.com/) |
| AutoGen | Agent对话 | 对话协作 | 多 Agent | [AutoGen](https://microsoft.github.io/autogen/) |
| DSPy | 程序化提示 | 自动优化 | 提示工程 | [DSPy](https://dspy.ai/) |
| Pydantic AI | 类型安全 | 类型安全 | Python | [Pydantic AI](https://ai.pydantic.dev/) |
| MetaGPT | 软件工程 | 完整开发流 | 代码生成 | [arXiv](https://arxiv.org/abs/2308.00352) |
